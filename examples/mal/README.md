# Ziku MAL Implementation

This directory contains the Ziku implementation of MAL (Make a Lisp).

## Structure

- `step0_repl.ziku`: Basic REPL
- `step1_read_print.ziku`: Reader and Printer
- `step2_eval.ziku`: Evaluator
- `step3_env.ziku`: Environments
- `step4_if_fn_do.ziku`: Conditionals and Functions
- `step5_tco.ziku`: Tail-call Optimization

## Running

Run the steps using the provided script (to be implemented):

```bash
./run step0_repl
```
